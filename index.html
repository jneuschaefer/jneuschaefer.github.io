Current Projects

Course Project (Advanced Machine Learning, Polytechnique/ENSAE)

Final project for the course Advanced Machine Learning at Inst. Polytechnique. The objective is to improve the adversarial robustness of convolutional neural networks in a medical image classification task (pneumonia detection from chest X-ray images). This is achieved by incorporating adversarial examples generated via the Fast Gradient Sign Method (FGSM) and Projected Gradient Descent (PGD) into the training process.
